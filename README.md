Dieses Projekt basiert auf einem bestehenden HTML-Prototyp und wird im Rahmen eines Lernprozesses schrittweise in eine moderne React-Anwendung mit TailwindCSS und TypeScript überführt. Ziel ist es, Best Practices für Frontend-Architektur, Responsivität und Wiederverwendbarkeit zu erlernen und anzuwenden – mit klar dokumentierten Arbeitsschritten.
